HTML

  • HTML stands for HyperText Markup Language.
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Semantic HTML is the preferred choice since it is easier to read by developers and it is more accessible to widest variety of users.
  • HTML uses a Parent-Child relationship. A parent element contains the child element.

CSS

  • CSS stands for Cascading Style Sheets.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Extrenal CSS is the preferred method of developers when adding CSS to HTML.
  • Class attributes can be assigned to an element in HTML, allowing us to freely choose which elements will be assigned to the CSS rule.
  • Colors in CSS can be assigned via semantic text or by using hexadecimal codes.

Git

  • git is a version control software.
  • git commands always begin with git.
  • git is used with Github so that we may collaborate in an Agile workflow.
  • git status: checks what branch we are currently on.
  • git checkout -b branch-name: creates a new branch and switches to it.
  • git add: adds a change in the working directory to the staging area.
  • git commit: captures a snapshot of your project's currently staged changes.
  • git pull: make sure branch is up to date.
  • git push: push to branch you are currenlty working in.
  • Github is used to complete pull request and publish to main branch.

JavaScript

  • JavaScript is a programming language that allows users to interact with websites.
  • Javascript is case sensitive!
  • A variable is a named container that allows us to store data in our code.
  • Variables are necessary, if values couldn't change then you couldn't do anything dynamic.
  • There are several types of operators that Javascript uses.
  • JS functions are a way of packaging functionality that you wish to reuse.
  • Control flow is the order in which a computer executes code in a script.
  • if Statements are a way of introducing decision-making into our code.
  • Arrays are a single variable that is used to hold a group of similar data, but arrays can contain multiple data types.
  • JS number system always begins at 0.
  • for Loops use predictable patterns of indices to perform a task on all array items by allowing a single code block to be executed over and over.
